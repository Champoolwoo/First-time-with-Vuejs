# Vuejs

###Install Vue
ข้อกำหนด : Node.JS เวอร์ชั้น 4.x ขึ้นไป
```bash
$ npm install -g vue-cli
```
###การใช้งาน
```bash
$ vue init <template-name> <ชื่อ DIR ที่จะสร้าง>
```
ตัวอย่าง
```bash
$ vue init webpack my-project
```
จากนั้นกด Enter ยาว
เสร็จแล้วเข้าไปที่ Dir ที่ได้สร้างไว้ แล้วใช้คำสั่ง
```bash
$ npm install
```
###Run
```bash
$ npm run dev
```
